# AXI/OBI conversion IPs

The repository contains SystemVerilog protocol converters between the PULP-platform [AXI](https://github.com/pulp-platform/axi) and [OBI](https://github.com/pulp-platform/obi) IPs.

They are designed by PULP-platform and are available under the Solderpad v0.51 license (See LICENSE).
